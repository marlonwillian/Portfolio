PROJETOS

Loja de Jogos

A Loja de Jogos é um projeto que utiliza React, Bootstrap e o Context API para simular uma plataforma de compra de jogos. Utiliza também o formato JSON para armazenar os jogos e suas informações, além de integrar o React Router para gerenciar as rotas do aplicativo.

Criador de Cartas

Criador de Cartas é um projeto utilizando HTML, CSS e Java Script que permite criar cartas personalizadas de jogadores de futebol, inspiradas no estilo visual dos jogos FIFA/FC. Esta aplicação proporciona a personalização de diversos elementos, como jogadores, times, posições e estatísticas.

Página inicial da Netflix

Este projeto é uma réplica da página inicial
da Netflix, desenvolvida com HTML5, CSS e JavaScript. Usando a biblioteca Owl Carousel para navegar intuitivamente pelos conteúdos. A combinação de tecnologias faz uma interface responsiva e visualmente atraente, capturando a essência da plataforma original.
